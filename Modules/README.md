# BlueHR Modules

Setiap fitur bisnis berada di folder module terpisah. Prinsip pengembangan:

- Core hanya untuk fondasi aplikasi.
- Fitur baru harus menjadi module baru atau extension module.
- Jangan mencampur logic antar module.
- Route, controller, model, view, migration, permission, settings dipisah per module.

Module final yang disiapkan:

- CoreHR
- Organization
- Documents
- Recruitment
- Onboarding
- Attendance
- Leave
- Overtime
- Approval
- Payroll
- FacilityBenefit
- Performance
